# Hangman Game

This project is a command-line Hangman Game developed with the coding language of Kotlin. 
The game allows the user to guess letters that are in a hidden word diplayed with underscores. 
It will track the attempts that the user guesses and will score and show the correct letters in the diplay. 
This program will demonstrate key programming and concepts like object oriented design and collections. 

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Install Java Developmet Kit
2. Install IntelliJ IDEA
3. Open IntelliJ and create a new Kotlin project
4. Copy and Paste the Hangman code into a .kt such as Main.kt
5. Click the Run button to run the program.

Instructions for using the software:

1. Enter the player's name when asked.
2. Choose a category to play in, tech or animals?
3. Select a differculty level, easy, medium, and hard?
4. Guess letters one at a time to show in the hidden word
5. Try to guess the word before running out of all the attempts given.
6. View your final score once you have run out of attempts or have guessed the hidden word.
